# 🐾 Adoption API

API REST desarrollada con Node.js y Express para la gestión de adopciones.  
Incluye tests funcionales, documentación con Swagger y dockerización completa.

---

## 🚀 Tecnologías utilizadas

- Node.js
- Express
- Jest + Supertest
- Swagger
- Docker
